Metodología / Methodology

🎯 Basado en Proyectos

Cada estudiante desarrolla un proyecto web personal a lo largo del semestre, aplicando conceptos de forma práctica e incremental.

📝 Un Commit por Clase

Estructura ágil adaptada a la educación: cada sesión termina con un commit que documenta el progreso semanal.

🤔 Codificación Crítica

Reflexión sobre el contexto social y ético de la tecnología, promoviendo diseño inclusivo y responsable.

♿ Accesibilidad por Defecto

Integración de principios de accesibilidad web desde el primer día, siguiendo estándares WCAG 2.1.
