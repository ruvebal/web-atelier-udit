# Evaluation & Assessment Hub

This hub centralizes the **evaluation philosophy, bilingual rationale documents, and LMS-ready artifacts** that power Web Atelier assessments. The resources below are language-agnostic, version-controlled, and aligned with the broader methodology described in the `/methodology` section.

## Available Resources

## Alignment with Methodology & Bibliography
- **Learning Cycle:** Evaluations measure the same Theory → Practice → Reflection triad described in `/methodology/*`.
- **Accessibility-by-default:** Rubrics reference WCAG, inclusive typography, and ethical-by-design checkpoints documented throughout `lessons/` and `references/`.
- **Evidence-based grading:** Rationale documents cite the canonical bibliography (`/bibliography`) to justify each assessment facet.
