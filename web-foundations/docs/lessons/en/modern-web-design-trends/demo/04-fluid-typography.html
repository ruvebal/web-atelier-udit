‚ôø 5. Accessibility in Typography

Inclusive design isn't optional‚Äîit's fundamental

Color Contrast

‚úì 7.5:1

Excellent Contrast

This black text on white has a ratio of 21:1, far exceeding the WCAG minimum of 4.5:1 for normal text.

‚úó 1.8:1

Insufficient Contrast

This light gray text is hard to read and doesn't meet accessibility standards. Avoid this in production.

Line Length

‚ùå Line Too Long: Extremely long lines make reading difficult because the eye has to travel a great horizontal distance, which tires the reader and makes it easy to lose one's place when moving to the next line. The rule is 60-80 characters per line.
‚úÖ Optimal Length: Lines of approximately 60-80 characters are ideal for reading. The eye can scan comfortably and move to the next line effortlessly. Use max-width to control this.
/* Accessibility Checklist */ /* 1. Sufficient contrast */ body { color: #1a1a1a; /* 4.5:1 minimum */ background: white; } /* 2. Adequate base size */ body { font-size: 1rem; /* 16px base, use rem */ } /* 3. Line length */ p { max-width: 75ch; /* 60-80 characters */ line-height: 1.6; /* 1.5-1.8 for body */ } /* 4. Respect user preferences */ /* Don't use fixed px - use rem so */ /* users can adjust font size */
Normal text: 4.5:1 Large text: 3:1 Base: 16px minimum Line: 60-80 chars

üåê 6. Web Considerations

Font stacks, loading strategy, and relative units

Robust Font Stacks

If the custom font doesn't load, system fonts take over:

1st Attempt: 'Inter' (Google Font)

2nd Fallback: -apple-system, Segoe UI (System)

3rd Fallback: sans-serif (Generic)

/* Complete Font Stack */ body { font-family: 'Inter', /* Custom font */ -apple-system, /* macOS San Francisco */ BlinkMacSystemFont, /* macOS Chrome */ 'Segoe UI', /* Windows */ Roboto, /* Android */ sans-serif; /* Generic */ } /* Font Loading Control */ @font-face { font-family: 'MyFont'; src: url('font.woff2') format('woff2'); font-display: swap; /* Show fallback immediately */ } /* font-display options: */ /* ‚Ä¢ swap: Text visible instantly ‚úì */ /* ‚Ä¢ block: Wait for font (FOIT risk) */ /* ‚Ä¢ fallback: Compromise between both */ /* ‚Ä¢ optional: Browser decides by connection */

Relative vs Fixed Units

‚ùå Fixed Units

p { font-size: 16px; } /* Doesn't respect user zoom */

‚úÖ Relative Units
