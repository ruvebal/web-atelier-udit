Plataforma E-commerce
Descripción del Proyecto

Plataforma completa de e-commerce construida con tecnologías modernas. Incluye catálogo de productos, autenticación de usuarios, carrito de compras y procesamiento de pagos.

Características Principales
  • Catálogo de productos con búsqueda y filtrado en tiempo real
  • Sistema de autenticación seguro con JWT
  • Integración con pasarela de pago (Stripe)
  • Panel de administración para gestión de inventario
  • Diseño responsivo para todos los dispositivos
Tecnologías Utilizadas

Frontend: React, Redux, Bootstrap
Backend: Node.js, Express, MongoDB
